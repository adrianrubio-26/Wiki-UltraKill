# V1 

## Descripción 
v1 es el personaje jugable principal de Ultrakill, un maquina suprema diseñado para acabar con la guerra de 1000 años eliminando a la otra maquina suprema conocida como THR-1000-Earthmover. 

esto no se llego a concretar ya que se firmo la paz y las maquinas dejaron de luchar y perdiendo su proposito v1 quedando desactivado.

## Apariencia

V1 es un robot humanoide azul con alas que le permiten hacer dashes a grandes velocidades saltar grandes distancias y guardar todas sus armas, aprte su armadura cuenta con un sistema de absorcion de sangre que le permite auto repararse

## Habilidades 
Aquí tienes un **resumen breve y compacto de las habilidades de V1**:

V1 es una máquina de combate que se cura absorbiendo sangre al atacar de cerca, lo que lo obliga a jugar de forma agresiva. Posee una movilidad extrema gracias a dashes, saltos aéreos y desplazamientos rápidos. Puede combinar armas y sus variantes para crear combos devastadores, además de usar monedas para redirigir disparos con precisión crítica. Cuenta con brazos especiales: uno para devolver ataques, otro para golpes explosivos y un gancho para controlar la distancia. Todo esto lo convierte en un arma altamente adaptable, rápida y letal, diseñada para sobrevivir solo mediante la destrucción constante.
